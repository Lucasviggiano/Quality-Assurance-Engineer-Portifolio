# Portfolio de Engenharia de Qualidade

Este repositorio contem o portfolio QA do Lucas Viggiano em formato de landing page estatica.

## Objetivo

Apresentar de forma clara os principais projetos de Quality Engineering publicados no GitHub, com foco em:
- automacao web
- automacao de API
- automacao mobile
- testes de performance
- CI/CD para qualidade continua

## Estrutura

- `index.html`: conteudo do portfolio e secoes da pagina
- `style.css`: visual, responsividade e componentes
- `script.js`: animacoes, menu mobile e validacao do formulario

## Como executar localmente

1. Abra esta pasta no VS Code.
2. Use Live Server (ou abra `index.html` no navegador).
